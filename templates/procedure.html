Procedure for Pre-Test:

  1. Fill the correct values in the table fields and click on 'Check answers' to check your response.

  2. After submission of a correct response, the 'Perform Simulation' button is activated and then you can move to the Simulation section.

Procedure for Simulation:

  1. The simulation tab consists of 2 sections.
  2. In the first section, the user needs to enter the correct Huffman codes for each character in the given string .


  3. On clicking the Check answer button, it shows whether the code submitted by the user is correct or not. Then the user may proceed to Section 2.
  4. In the second section, the user needs to select the correct input sequence from the dropdown such that the given output is obtained. On submission of correct answer, the user will be allowed to move to the Post-Test section.

  5. Click on 'Perform Post-test' to move to the next section.
